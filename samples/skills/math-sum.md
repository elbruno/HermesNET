# Math Sum

Given a list of numbers, calculate their sum.

## Input
A comma-separated list of numbers: `3, 5, 7`

## Output
The sum as a single number: `15`

## Example
Input: `10, 20, 30, 40`
Output: `100`

## Notes
- Handles negative numbers: `-5, 10, 3` → `8`
- Handles decimals: `1.5, 2.5, 3` → `7`
