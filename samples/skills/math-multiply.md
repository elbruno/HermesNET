# Math Multiply

Calculate the product of multiple numbers.

## Input
Comma-separated numbers: `2, 3, 4`

## Output
Product: `24`

## Example
Input: `5, 6, 2`
Output: `60`

## Notes
- Handles negative numbers: `-2, 5` → `-10`
- Handles decimals: `1.5, 4` → `6`
- Multiplying by zero always returns `0`
